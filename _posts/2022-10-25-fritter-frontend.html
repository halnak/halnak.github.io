GitHub for this project

This is for an ongoing project in Software Studio (6.170/6.1040), the advanced software consutrction class I am taking this semester. This implementation is the front-end for my Fritter site, a spinoff of Twitter which emphasizes community through the use of multiple Fritter Circles: membership circles for which users can join/be added to and then post to for a targeted feed.

This project is ongoing and I am continuing to update it. The frontend portion in particular is currently being built, and is unfinished as of 10/25/2022. The full frontend assignment will be completed on 11/06/2022.
